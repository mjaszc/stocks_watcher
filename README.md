# Stocks watcher

Stock performance tracking application with daily database updates and interactive [Plotly.js](https://github.com/plotly/plotly.js) charts. Monitor and analyze market movements across various timeframes to support informed investment decisions.

## Tech Stack

**Frontend:** Vite, React, PrimeReact, Plotly.js

**Backend:** FastAPI, PostgreSQL, Alembic, Pydantic, Docker, Pandas
